# Web back-end (1) - Một số khái niệm

## Một số khái niệm
### 1.1 Web là gì?
Web (hay còn gọi là WWW - World Wide Web) được định nghĩa là một mạng lưới thông tin toàn cầu. Hệ thống này hoạt động dựa trên nền tảng hạ tầng là Internet và được cấu thành từ vô số các trang web (webpage) kết nối chặt chẽ với nhau. Thông qua các liên kết (link), người dùng có thể di chuyển từ trang web này sang trang web khác để tiếp cận nguồn dữ liệu đa dạng từ văn bản, hình ảnh, âm thanh cho đến video
